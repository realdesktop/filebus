
Минимальный набор объектов, необходимый для функционирования шины.


IxpServer - корневой объект IXP (9P) сервераw


RbusRoot (wpas_rbus_priv) - корневой объект шины.
    содержит dial string для регистрации сервера
    содержит указатель на корневой объект сервера
    содержит указатель на объект шины (Rbus)


Rbus (IxpFileIdU? ) - объект шины (множество) 
    содержит корневую очередь событий ( IxpPending )
?   содержит имя типа
?   содержит путь
    содержит набор свойств


Корневой нативный объект приложения (wpa_global)
    содержит указатель на корневой объект шины ( -> rbus )


Нативный объект приложения (wpa_supplicant - объект сетевого интерфейса)
    содержит указатель на корневой нативный объект приложения
    содержит указатель на объект шины (Rbus)
